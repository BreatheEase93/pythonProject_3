# Учебное приложение

## Описание
Простое учебное приложение для работы с товарами и категориями.

## Текущая реализация

### Классы
1. **Product** - класс для представления товара
2. **Category** - класс для представления категории товаров
3.**Smartphone** - класс наследник класса Product: «Смартфон»
4.**LawnGrass** - класс наследник класса Product: «Трава газонная»
4.**Order** - класс для представления заказа
4.**ProductReprMixin** - миксин для печати информации о создании объекта
